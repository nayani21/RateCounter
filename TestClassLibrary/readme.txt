* This is a awesome library can be used to rate your applications using stars by default.
  you can change the star symbol as your prefernce.

* when you install  the RateCounter library you should be able to install only to the portable project.

* If you want to use a image instead of default stars image,you have to put the image in Resource folder(android).

* When you access RateCounter library please consider about the width and height.Default image width and height ratio is 129:30.
  if you use a coustom image you have to set the width and height according to the coustom image ratio.
